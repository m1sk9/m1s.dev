This project uses [Deno](https://deno.land) 🦕 and [fresh](https://fresh.deno.dev/) 🍋.

To start it locally, execute the following command:

```sh

# --- m1s.dev use Deno 2 :)
curl -fsSL https://deno.land/install.sh | sh

deno task start
```
